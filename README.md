# Интерактивный дашборд ряда Фурье 

Это интерактивный веб - дашборд на **Streamlit**, который выполняет аналитическое разложение произвольной заданной функции в тригонометрический ряд Фурье и визуализирует этот процесс в реальном времени

## Математическая модель для ряда Фурье 

Программа выполняет разложение функции $f(x)$ на интервале $[x_{min}, x_{max}]$ для произвольного полного периода $T = x_{max} - x_{min}$:

$$f(x) \approx \frac{a_0}{2} + \sum_{n=1}^{\infty} \left( a_n \cos\left(\frac{2n\pi x}{T}\right) + b_n \sin\left(\frac{2n\pi x}{T}\right) \right)$$
            
Движок рассчитывает амплитуды гармоник через определенные интегралы:

$$a_0 = \frac{2}{T} \int_{x_{min}}^{x_{max}} f(x) \, dx$$
               
$$a_n = \frac{2}{T} \int_{x_{min}}^{x_{max}} f(x) \cos\left(\frac{2n\pi x}{T}\right) \, dx$$

$$b_n = \frac{2}{T} \int_{x_{min}}^{x_{max}} f(x) \sin\left(\frac{2n\pi x}{T}\right) \, dx$$
             

## Возможности проекта

1. Парсинг любой произвольной функции в текстовое поле 
2. Гибрид технологий для вычисления функции приближения
3. Анимация приближения функции через тригонометрический ряд
4. Встроенная математическая справка

## Запуск

1. Клонируйте репозиторий проекта 
```bash
git clone https://github.com
cd fourier-dashboard
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```
3. Запустите дашборд:
```bash
streamlit run app.py
```
